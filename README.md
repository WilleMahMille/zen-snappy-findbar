# Snappy Findbar

A Zen Browser mod that transforms the findbar into a sleek, centered overlay at the top of your browser with smooth animations and a minimalistic design.

## Features

- **Centered positioning** - Findbar appears at the top center of the viewport
- **Smooth animations** - Snappy slide-in/slide-out transitions
- **Minimalistic design** - Cleaner checkbox styling with background highlights instead of icons
- **Hidden status messages** - Removes redundant status text for a more compact layout
- **Fully customizable** - Adjust colors, positioning, and animation timing through preferences

## Configurable Variables

All settings can be customized through Zen Browser's mod preferences UI:

### Animation Settings

- **Animation Duration** (default: `0.2s`)
  - Controls how fast the findbar appears and disappears
  - Examples: `0.2s`, `300ms`, `0.5s`

- **Animation Easing** (default: `ease`)
  - Controls the animation timing function
  - Examples: `ease`, `ease-in-out`, `cubic-bezier(0.4, 0, 0.2, 1)`

### Position Settings

- **Top Position** (default: `5vh`)
  - Distance from the top of the viewport
  - Examples: `5vh`, `50px`, `10%`

### Visual Settings

- **Background Color** (default: `#000000`)
  - Background color of the findbar
  - Color picker available in preferences

- **Border Radius** (default: `15px`)
  - Roundness of the findbar corners
  - Examples: `15px`, `1rem`, `8px`

- **Border Color** (default: `#808080`)
  - Color of the findbar border
  - Color picker available in preferences

## Installation

1. Install via the [Zen Mods Store](https://zen-browser.app/mods/)
2. Customize settings in Zen Settings → Mods → Snappy Findbar
